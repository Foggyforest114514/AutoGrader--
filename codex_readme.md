2.AutoGrader作业代码自动评分系统，项目编号：Beta
    主要功能：
        1.学生信息管理
        2.评分查询
        3.自动评分
        4.测评代码与分析
        5.题库管理
        6.前后端分离，BS架构

    业务逻辑：
        1.导入学生名单，按课程编号，课序号分班管理
        2.显示班级评分结果，提供本人评分结果详细信息查询
        3.管理待测评代码，包含接收，存储，测评，测评结果存储分析
        4.题库管理：针对不同题型，实现判断逻辑框架，生成测试代码
        5.用户管理

    开发工具：
        1.Python：FastAPI
        2.JavaScript：Nodejs
        3.沙盒：Firejail，docker等

    功能模块：
        UI，模块编号，模块编号：B-1
            同学信息管理，导入pdf，xlsx格式同学信息，包含姓名、学号、课程编号、课序号等
            同学信息编辑功能，操作需要权限控制
            成绩显示，按班级、按课程显示课程成绩，提供查询功能
            作业答案提交，接收同学提交的作业答案，使用API接口发送到B-2模块

        代码测评，模块编号：B-2
            由API接口接收B-1模块发送的待测作业代码
            由电子邮件接收待测作业代码
            存储管理待测作业代码
            检查代码的静态合法性，记录评价结果，包含总体评价与细节评价
            测试代码的动态运行逻辑准确性，记录评价结果，包含总体评价与细节评价
            评价过程与结果记录到数据库，按格式导出

        题库管理，模块编号：B-3
            待测代码安全性确认
            实现以下类型题目测试框架：
                1.待测代码由命令行接收测试数据
                2.待测代码由文件读取测试数据
                3.待测代码实现了某种调用接口，需要经过使用import、include等操作后才能调用并测试

            题库维护
            生成答案代码评价结果，包含总体评价与细节评价

        后端数据库管理维护：模块编号：B-4
            维护数据库的正常运行
            维护消息中间件运行（按需）
            定义数据库表结构
            定义管理各模块间API调用接口
            实现用户管理API接口


    数据来源：
        待测试代码提交方式：电子邮件，网页提交
